# plattenwascher-firmware

## reqirements:

* python >= 3.7
* nodejs >= 12.x.x
* pipenv 

create a file called 'src/firmware/wlan.json' of the following form 
```json
{
    "ssid": "your_ssid",
    "password": "your_password"
}
```
there is also an example file called 'src/firmware/wlan.json.example'

```shell
$ pipenv install
```

```shell
$ pipenv shell
```

```shell
$ npm install
```

```shell
$ npm run build
```
